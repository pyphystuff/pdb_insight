# 🔬 PDBInsight: Terminal-Based Protein Structural Analyzer

**PDBInsight** is a lightweight CLI tool designed for quick protein structure analysis and visualization directly in the terminal. It provides researchers with a fast way to inspect PDB files, sequences, and secondary structures without leaving the command line.

## ✨ Features

* **3D ASCII Viewer**: Interactive rotation of the Alpha Carbon ($C\alpha$) backbone.
* **1D SS-Vis**: Aligned sequence and secondary structure mapping (Helices, Sheets, Loops).
* **Structure Stats**: Quick extraction of metadata, chain information, and amino acid composition.
* **AI Integration**: Automated structural summaries using LLMs via Ollama.
